Detailed Description

Adds a rotation to attached Gameobject.

Detailed

Using Vector3 coordinates, the gameobject is rotateted based on the speed and time passed. If the speed is increased or decreased the rotation will be affected.

Note
Be careful when placing script on Gameobjects with collider boxes as it may push or block other Gameobjects. In these instances be sure to have IsTrigger activated so the collider doesn't cause issues
