Function used to increase or decrease the Player's score.

Detailed

This function is what governs the points gained/lost in 3 categories. Those categories are Professionalism , Customer Service, and Technical Skills.

Parameters
profThis variable represents the points gained or lost in the "Profesionalism" category
csThis variable represents the points gained or lost in the "Customer Service" category
techThis variable represents the points gained or lost in the "Technical Skills" category
correctThis variable signifies whether the character completed the task correctly
playSoundThis variable signifies if there should be an audioclip played for the action
