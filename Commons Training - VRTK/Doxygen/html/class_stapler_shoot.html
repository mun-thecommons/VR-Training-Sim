Detailed Description

Script controlling the motion and logic of the stapler

Detailed

Changes the vector location of the stapler object each frame update and also determines what happens during a collision with another object based upon the object

void StaplerShoot.OnTriggerEnter ( Collider  other)
private

Detects what the stapler collides with.

Detailed

During a collision the scripts checks the tag of the Gameobject collided with. If the collision is with a Monitor or Printer they will be destroyed and a White Smoke prefab will instantiate in its place.

If it is a client then it will decrement a scorepoint. If it is an alien then it will trigger the "Alien Shot" animation.
