Detailed Description

Contains the logic used for the Cashbox Client, and how it interacts with the CampusCard and CashBox

Detailed

This script will interact with both the CampusCard and CashBox scripts so that the response of the Client is done correctly. The Renderer of the Campus Card attached to the NPC is initially set to inActive so that when the player interacts with them it has the effect of being "Dropped" by the NPC. Once the Campus Card is returned it will go back to disabling the Renderer to make it appear as if the NPC took their card back.
