Detailed Description

Controls facial expression of UMA Models

Detailed

This script directly interacts with the adjustment nodes for the UMA Model facial expressions. This is used for creating Happy, Angry, Sad facial expressions which are used within other scripts when answering questions correctly

void UMAMoodController.OnCreated ( UMAData  data)

Enables facial moving elements upon creation.

Detailed

Turns on blinking and saccades from the UMA options. Blinking will allow the models to periodically blink their eyes, while saccades will allow there eyes to move around, to give the impression they are staring at different things.
